# branch

Branch: 분기점을 생성하여 독립적으로 코드를 변경할 수 있도록 도와주는 모델

```shell
생성 $ git branch {name}
이동 $ git checkout
이동 $ git switch
병합 $ git merge
삭제 $ git branch -D
```
